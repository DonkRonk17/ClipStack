## 1. TITLE CARD (16:9 Landscape - 1920x1080px)

**For:** GitHub repository header image, presentations, documentation

**DALL-E Prompt:**
```
Create a sleek, modern title card for "ClipStack" - a clipboard history manager for power users.

STYLE:
- Dark navy blue background (#0a0e27)
- Bright cyan accents (#00d4ff)
- Tech-forward, professional aesthetic
- Minimalist geometric patterns

LAYOUT:
- Large bold title "ClipStack" centered, modern sans-serif font
- Smaller tagline below: "Never lose a copied snippet again"
- Visual element: Stack of translucent clipboard documents or cards, layered at angles
- The stack shows multiple text snippets fading from sharp (recent) to subtle (older)
- Subtle circuit board or data flow patterns in background
- A search icon and history icon subtly integrated

MOOD: Professional, efficient, organized, developer-focused, Team Brain ecosystem

NO text except "ClipStack" and the tagline. Clean, minimal, powerful.
```

## Beacon HQ Visual Style Reference

**Brand Colors:**
- Primary: Dark Navy (#0a0e27)
- Accent: Bright Cyan (#00d4ff)
- Supporting: Deep Purple (#1a0d2e), Teal (#00fff5)
- Text: White (#ffffff), Light Gray (#e0e0e0)

**Typography:**
- Modern sans-serif (Roboto, Inter, SF Pro)
- Bold weights for titles
- Clean, readable

**Visual Elements:**
- Geometric shapes
- Layered/stacked elements (fits clipboard history theme)
- Data visualization hints
- Minimal gradients (subtle when used)

**Mood:**
- Professional
- Organized
- Efficient
- Developer-focused
- Part of unified Team Brain ecosystem
